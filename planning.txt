...........Routes..............
User Authentication:
     /signup:POST - Create a new user account.
     /login: POST - log in to an exiting account. [aadhar card or password]


Voting:
      /candidates: GET - get the list of canditates.
      /vote/:candidatesID :POST -vote for a specific candidate.


Vote Counts:
       /vote/counts:GET- show all counting on live.


User Profile:
      /profile: GET- get the user's profile information.
      profile/password: POST - change user's  password     


Admin candidate Managment:
      /candidates :POST- create a new candidate
      /candidates/:candidatesID: PUT -update an exiting candidate.
      /candidates/:candidatesID: DELETE - delete a candidate from the list.
